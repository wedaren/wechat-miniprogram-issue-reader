# 图标资源说明

本目录需要包含以下图标文件：

## 必需的图标文件

1. **tree.png** / **tree-active.png**
   - 问题总览Tab图标
   - 建议使用树状结构图标
   - 尺寸：44x44px

2. **star.png** / **star-active.png**  
   - 关注问题Tab图标
   - 建议使用星星或心形图标
   - 尺寸：44x44px

3. **orphan.png** / **orphan-active.png**
   - 孤立问题Tab图标  
   - 建议使用收件箱或孤岛图标
   - 尺寸：44x44px

4. **recent.png** / **recent-active.png**
   - 最近问题Tab图标
   - 建议使用时钟图标
   - 尺寸：44x44px

## 图标规范

- **格式**: PNG
- **尺寸**: 44x44px（推荐） 
- **背景**: 透明
- **颜色**: 
  - 普通状态：#999999
  - 选中状态：#007aff

## 获取图标的方式

1. **免费图标库**:
   - [Iconify](https://iconify.design/)
   - [Feather Icons](https://feathericons.com/)
   - [Heroicons](https://heroicons.com/)

2. **设计工具**:
   - Figma
   - Sketch  
   - Adobe Illustrator

3. **在线生成**:
   - [Favicon Generator](https://favicon.io/)
   - [Icons8](https://icons8.com/)

## 临时解决方案

在没有真实图标的情况下，可以：

1. 使用emoji作为临时图标
2. 使用纯色矩形占位
3. 使用系统字体图标

注意：正式发布前必须替换为正式的图标文件。
